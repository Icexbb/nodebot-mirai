# nodebot-mirai
Use nodejs to write mirai bot, support websocket and http api

## Deploy

``` bash
    git clone https://github.com/icexbb/nodebot-mirai
    cd nodebot-mirai
    npm i
    # Create run.ts and Create a NodeBot Object with mirai-http-api config
    npm run start
```
